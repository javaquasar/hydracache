## 0.12.0 Implementation Notes

Started HydraCache `0.12.0` with reusable database query cache policies:

- Added `QueryCachePolicy` in `hydracache-db`.
- Moved `DbQuery` internals to the policy object.
- Added `DbCache::cached_with::<T>(policy)`.
- Added `DbQuery::with_policy(policy)` and `DbQuery::cache_policy()`.
- Added `DbQuery::collection(name)`.
- Added `DbQuery::load(...)` as a repository-style alias for
  `fetch_with(...)`.
- Re-exported `QueryCachePolicy` from `hydracache-sqlx`.
- Added tests for policy metadata, reuse, replacement, collection policy, SQLx
  re-export, and the new `load(...)` cache-hit path.
