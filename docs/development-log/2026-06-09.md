# 2026-06-09

## 0.16.0 Sandbox Lab Hardening

Extended the workspace-only `hydracache-sandbox` from a manual API playground
into a reproducible cache behavior lab:

- Added named scenario runner presets for golden path, TTL, single-flight,
  invalidation race, negative suite, and self-test.
- Added flow timelines keyed by `flow_id` so one operation can be inspected as
  an ordered event sequence.
- Added latency summaries to event logs, reports, exports, timelines, scenario
  responses, and manual benchmark responses.
- Added local profile comparison for memory, SQLite memory, and SQLite file,
  with Postgres profiles reported as skipped to avoid implicit external
  dependencies.
- Added replay, fault-injection, and manual benchmark endpoints.
- Added an optional local bearer-token guard through
  `HYDRACACHE_SANDBOX_TOKEN` or `--token`.
- Updated the no-CDN dashboard, HTTP collection, PowerShell demo flow, README,
  release notes, publishing notes, and observability plan.
- Added route-level tests for the new sandbox lab endpoints and auth guard.

## 0.17.0 Sandbox Scenario Documents

Started the `0.17.0` sandbox lab iteration with executable scenario documents
and richer release-check tooling:

- Added JSON and small-YAML scenario documents with step-level assertions.
- Added benchmark comparison, event-session import, Prometheus-style metrics,
  OpenTelemetry-style trace demo output, DB seed reports, and OpenAPI
  generated-client checks.
- Added committed scenario recipes, migration/seed assets, a minimal generated
  fetch client, README notes, HTTP requests, and PowerShell demo-flow coverage.

## 0.17.0 Sandbox Regression Lab Extension

Expanded the same `0.17.0` sandbox release into a more reproducible regression
and teaching lab:

- Added committed scenario file and suite runners for JSON/YAML recipes.
- Added timeline assertions and richer scenario pass/fail reporting.
- Added flow catalog and retained-flow replay endpoints.
- Added seeded products and order summaries to the query-cache demo surface.
- Added dashboard controls for scenario files, suites, flow replay, product and
  order query-cache demos, benchmark diff, and generated-client smoke.
- Improved benchmark reports with loader-call ratio, operation latency
  percentiles, p95 diff, and compact comparison verdicts.
- Added OpenAPI generated-client smoke checks and expanded the committed fetch
  client fixture.
- Updated README, publishing notes, Compose docs, HTTP collection, demo script,
  release notes, and sandbox route tests.
